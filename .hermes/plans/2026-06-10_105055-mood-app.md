# Moodly — 极简情绪记录 App 实施方案

> **For Hermes:** 用 subagent-driven-development 逐任务派发,每任务做完整 TDD 闭环 + 两轮 review。

**Goal:** 一款极简情绪记录 App。上半屏是类 GitHub 贡献图风格的近 30 天日历,下半屏是 8 个情绪按钮,长按打卡,色块按次数加深。

**Architecture:**
- 单页 Expo (React Native) 应用,Expo Router 单路由栈
- `expo-sqlite` 本地存储打卡记录(日期 + 情绪 + 时间戳),所有聚合查询走 SQL
- 全局状态用 Zustand 单 store,避免 prop drilling
- 震动反馈:`expo-haptics` 的 `Haptics.impactAsync(Medium)`
- 日历用纯 View 网格 + StyleSheet,无第三方日历库(避免版本地雷,本场景只需要网格)

**Tech Stack:**
- Expo SDK 52 + React Native 0.76 + TypeScript (strict)
- expo-router 4.x
- expo-sqlite 15.x
- expo-haptics 14.x
- zustand 5.x
- Vitest + @testing-library/react-native(单测数据层和 store)
- ESLint + Prettier(expo 默认 preset)

**关键决策(已与用户对齐):**
- 技术栈:Expo (TypeScript + Expo Router)
- 日历跨度:近 30 天(纵向网格,5 列 × 6 行 或 6 列 × 5 行,见 Task 6)
- 按钮排布:8 个一排 × 1 行(横向滑动 overflow-x:auto 风格)
- 情绪体系:8 种固定颜色(详见下方"情绪调色板")

## 一、情绪调色板(写在 `src/constants/emotions.ts`)

| key | 中文 | 英文 | 颜色 |
|---|---|---|---|
| `happy` | 开心 | Happy | `#F5C518` |
| `calm` | 平静 | Calm | `#7FBF7F` |
| `grateful` | 感恩 | Grateful | `#E89AAE` |
| `excited` | 兴奋 | Excited | `#E89B4A` |
| `anxious` | 焦虑 | Anxious | `#8B6FBF` |
| `sad` | 难过 | Sad | `#4A6FA5` |
| `angry` | 愤怒 | Angry | `#D64545` |
| `tired` | 疲惫 | Tired | `#8A9BA8` |

排序逻辑:从高愉悦高能量到低愉悦低能量,布局时左侧偏正向、右下偏负向,符合情绪地理直觉。

**打卡深度(次数 → 透明度):**
- 0 次:空格(`#F2F2F2` 极浅灰)
- 1 次:`rgba(颜色, 0.18)`
- 2 次:`rgba(颜色, 0.45)`
- 3 次:`rgba(颜色, 0.78)`
- ≥4 次:`rgba(颜色, 1.0)`(封顶)

**单情绪模式**:日历显示该情绪的分布,空格 = 没记录。

**聚合模式(默认)**:每天哪个情绪次数最多就展示哪个;若平局,按"调色板顺序"取第一个;格子深度 = 该日总打卡次数(1-4+ 级)。

```typescript
const EMPTY = '#F2F2F2';

export function hexToRgb(hex: string): [number, number, number] {
  const h = hex.replace('#', '');
  return [parseInt(h.slice(0,2),16), parseInt(h.slice(2,4),16), parseInt(h.slice(4,6),16)];
}

export function intensityToRGBA(hex: string, count: number): string {
  if (count <= 0) return EMPTY;
  const alpha = count >= 4 ? 1 : [0, 0.18, 0.45, 0.78, 1][count];
  const [r,g,b] = hexToRgb(hex);
  return `rgba(${r},${g},${b},${alpha})`;
}
```

```typescript
export function toDateKey(d: Date): string {
  const y = d.getFullYear();
  const m = String(d.getMonth()+1).padStart(2,'0');
  const day = String(d.getDate()).padStart(2,'0');
  return `${y}-${m}-${day}`;
}

export function last30Days(today: Date = new Date()): string[] {
  const out: string[] = [];
  for (let i = 29; i >= 0; i--) {
    const d = new Date(today);
    d.setDate(today.getDate() - i);
    out.push(toDateKey(d));
  }
  return out;
}
```

```typescript
import * as SQLite from 'expo-sqlite';

const DB_NAME = 'moodly.db';

let _db: SQLite.SQLiteDatabase | null = null;

export async function getDB(): Promise<SQLite.SQLiteDatabase> {
  if (_db) return _db;
  _db = await SQLite.openDatabaseAsync(DB_NAME);
  await _db.execAsync(`
    CREATE TABLE IF NOT EXISTS mood_entries (
      id INTEGER PRIMARY KEY AUTOINCREMENT,
      date TEXT NOT NULL,
      emotion TEXT NOT NULL,
      created_at INTEGER NOT NULL
    );
    CREATE INDEX IF NOT EXISTS idx_mood_date ON mood_entries(date);
  `);
  return _db;
}

export function __setDB(db: SQLite.SQLiteDatabase | null) {
```

```typescript
import { getDB } from './database';
import type { EmotionKey, MoodEntry } from '../types';

export async function addEntry(emotion: EmotionKey, date: string, createdAt = Date.now()): Promise<void> {
  const db = await getDB();
  await db.runAsync(
    'INSERT INTO mood_entries (date, emotion, created_at) VALUES (?,?,?)',
    date, emotion, createdAt
  );
}

export async function deleteEntry(id: number): Promise<void> {
  const db = await getDB();
  await db.runAsync('DELETE FROM mood_entries WHERE id = ?', id);
}

export async function getEntriesByDateRange(start: string, end: string): Promise<MoodEntry[]> {
  const db = await getDB();
  const rows = await db.getAllAsync<any>(
    'SELECT id, date, emotion, created_at as createdAt FROM mood_entries WHERE date BETWEEN ? AND ? ORDER BY created_at ASC',
    start, end
  );
  return rows.map(r => ({ id: r.id, date: r.date, emotion: r.emotion, createdAt: r.createdAt }));
}
```

### Task 5: 日历数据 hook(把 repository 输出转 CalendarCell[])

**目标:** 30 天 × 8 情绪 → 渲染就绪的网格数据。

**新建:** `src/hooks/useCalendarData.ts`
```typescript
import { useEffect, useState, useCallback } from 'react';
import { last30Days } from '../utils/date';
import { getEntriesByDateRange, addEntry as repoAdd } from '../db/repository';
import { EMOTIONS } from '../constants/emotions';
import type { CalendarCell, EmotionKey } from '../types';

export function useCalendarData() {
  const [cells, setCells] = useState<CalendarCell[]>([]);
  const [loading, setLoading] = useState(true);

  const refresh = useCallback(async () => {
    setLoading(true);
    const days = last30Days();
    const entries = await getEntriesByDateRange(days[0], days[days.length-1]);
    const next = days.map<CalendarCell>((date) => {
      const perEmotion = EMOTIONS.reduce((acc, e) => ({ ...acc, [e.key]: 0 }), {} as Record<EmotionKey, number>);
      let total = 0;
      let dominant: EmotionKey | null = null;
      let dominantCount = 0;
      for (const entry of entries.filter(e => e.date === date)) {
        perEmotion[entry.emotion]++;
        total++;
        if (perEmotion[entry.emotion] > dominantCount) {
          dominantCount = perEmotion[entry.emotion];
          dominant = entry.emotion;
        }
      }
      const cappedCount = Math.min(total, 4);
      return { date, count: cappedCount, emotion: dominant, perEmotion };
    });
    setCells(next);
    setLoading(false);
  }, []);

  useEffect(() => { refresh(); }, [refresh]);

  const addEmotion = useCallback(async (emotion: EmotionKey) => {
    const today = last30Days()[29];
    await repoAdd(emotion, today);
    await refresh();
  }, [refresh]);

  return { cells, loading, refresh, addEmotion };
}
```

```tsx
}
```

### Task 7: 进度环 + 长按手势 hook

**目标:** `ProgressRing` 用 SVG 画圆环;`useLongPress` 返回 `onPressIn`/`onPressOut`。

**新建:**

`src/components/ProgressRing.tsx`
```tsx
import React from 'react';
import { View, StyleSheet } from 'react-native';
import Svg, { Circle } from 'react-native-svg';

interface Props {
  size: number;
  strokeWidth: number;
  progress: number;       // 0..1
  color: string;
  children?: React.ReactNode;
}

export function ProgressRing({ size, strokeWidth, progress, color, children }: Props) {
  const r = (size - strokeWidth) / 2;
  const c = 2 * Math.PI * r;
  const dashOffset = c * (1 - Math.max(0, Math.min(1, progress)));
  return (
    <View style={{ width: size, height: size, alignItems: 'center', justifyContent: 'center' }}>
      <Svg width={size} height={size} style={StyleSheet.absoluteFill}>
        <Circle
          cx={size/2} cy={size/2} r={r}
          stroke={color} strokeWidth={strokeWidth} fill="none"
          strokeDasharray={`${c} ${c}`} strokeDashoffset={dashOffset}
          strokeLinecap="round"
          transform={`rotate(-90 ${size/2} ${size/2})`}
        />
      </Svg>
      {children}
    </View>
  );
}
```

`src/hooks/useLongPress.ts`
```typescript
import { useRef } from 'react';

export interface LongPressOptions {
  duration: number;        // ms
  onComplete: () => void;  // 满时触发
  onProgress?: (p: number) => void; // 0..1
}

export function useLongPress({ duration, onComplete, onProgress }: LongPressOptions) {
  const timer = useRef<NodeJS.Timeout | null>(null);
  const start = useRef<number>(0);
  const fired = useRef(false);

  const clear = () => {
    if (timer.current) { clearInterval(timer.current); timer.current = null; }
  };

  return {
    onPressIn: () => {
      fired.current = false;
      start.current = Date.now();
      clear();
      const tick = () => {
        const elapsed = Date.now() - start.current;
        const p = Math.min(1, elapsed / duration);
        onProgress?.(p);
        if (p >= 1) {
          fired.current = true;
          onComplete();
          clear();
        } else {
          timer.current = setTimeout(tick, 16);
        }
      };
      timer.current = setTimeout(tick, 0);
    },
    onPressOut: () => {
      clear();
      onProgress?.(0);
    },
    completed: () => fired.current,
  };
}
```

- 提前 400ms 松开 `onProgress(0)` 被调用,`onComplete` 不触发

```tsx
export function MoodButton({ emotionKey, size, onRecord }: Props) {
  const e = EMOTION_MAP[emotionKey];
  const [progress, setProgress] = useState(0);
  const selected = useMoodStore(s => s.selectedEmotion);
  const setSelected = useMoodStore(s => s.setSelectedEmotion);
  const lp = useLongPress({
    duration: 800,
    onProgress: setProgress,
    onComplete: async () => {
      await onRecord(emotionKey);
      Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Medium).catch(() => {});
      setProgress(0);
    },
  });

  return (
    <Pressable
      onPress={() => setSelected(emotionKey)}
      onPressIn={lp.onPressIn}
      onPressOut={lp.onPressOut}
      style={({ pressed }) => [
        styles.touch,
        { width: size, height: size },
        pressed && { opacity: 0.85 },
      ]}
    >
      <ProgressRing size={size} strokeWidth={4} progress={progress} color={e.color}>
        <View style={[
          styles.square,
          {
            backgroundColor: e.color,
            borderColor: selected === emotionKey ? '#222' : 'transparent',
          },
        ]} />
      </ProgressRing>
    </Pressable>
  );
}

const styles = StyleSheet.create({
  touch: { padding: 4, alignItems: 'center', justifyContent: 'center' },
  square: { width: '70%', height: '70%', borderRadius: 14, borderWidth: 2 },
});
```

3. **`expo-sqlite` 在 Expo Go 中可能不可用** — 如出现,改用 `expo install expo-sqlite` 后跑 `expo prebuild`
4. **日历格大小写死 44** — 大屏密集,但极简优先;后续可加尺寸档位
5. **没做导出/通知** — YAGNI,v2 再加
6. **平局时按 order 取第一个** — 简单稳定,如果用户要"混合色"格子再加
